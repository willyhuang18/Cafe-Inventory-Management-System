# Cafe-Inventory-Management-System
Brew.Base — Cafe Inventory Management System
A web application to help small and medium-sized cafe owners digitize and streamline the management of their menus and inventories.
Authors

Pangta Huang — Menu Management
Hsin-Yao Huang (Amy) — Inventory Management

## Setup guide
1. Clone the repo
2. `npm install`
3. Copy `.env.example` to `.env` and fill in your MongoDB credentials
4. `npm run seed` to populate the database
5. `npm run dev` to start the dev server
6. Open http://localhost:3000
